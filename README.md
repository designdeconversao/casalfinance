# CasalFinance 💑

Sistema de gestão financeira para casais, com foco em objetivos compartilhados e carteiras digitais.

## Tecnologias
- Next.js 14
- Prisma + PostgreSQL (Neon)
- Recharts
- Canvas Confetti

## Configuração
1. Instale as dependências: `npm install`
2. Configure o `.env` com a URL do PostgreSQL
3. Gere o cliente Prisma: `npx prisma generate`
4. Rode as migrações: `npx prisma db push`
5. Inicie o servidor: `npm run dev`
